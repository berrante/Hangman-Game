# Hangman-Game
## Description
 
A classic hangman-style game, for cat people. =^..^=


## Technologies Used
* HTML/CSS
* Javascript


## Gameplay
Game begins with the following default parameters:
* Wins : 0
* Randomly generated "Current Word" that must be guessed
* Guesses Remaining: 10
* Letters Guessed: blank
